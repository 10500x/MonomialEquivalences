# Instalación
Para correr los códigos hay que tener instalado el sagemath. Para instalarlo, dirijase a
```
https://www.sagemath.org
```
# Uso
en caso del .ipynb hay que utilizar los notebooks de jupyter para correrlos. Para ello, debe dirigirse a la terminal y escribir
```
sage -n jupyter
```
O bien abir el sagemath con el acceso directo. De ahí, elegir el archivo, elegir el kernel de sagemath y correrlo.
En caso del .py, se lo puede llamar directamente desde la terminal.
```
sage equivalenciaNoPrimoNuevoFinal.py
```
